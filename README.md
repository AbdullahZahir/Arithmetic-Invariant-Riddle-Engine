# Arithmetic-Invariant-Riddle-Engine
The Arithmetic Invariant Riddle Engine is a Java application that demonstrates mathematical invariance through a series of deterministic arithmetic transformations. The program accepts user-provided numeric input, applies predefined operation sequences, and consistently produces a fixed output value independent of the starting number. Multiple riddle configurations are implemented, including algorithms that always converge to 3 and 7, illustrating the underlying algebraic properties of invariant transformations.

The application incorporates user-input validation, supports both integer and floating-point values, and includes an automated testing suite composed of positive, negative, zero, and decimal test cases to verify correctness across diverse numerical scenarios. The implementation leverages procedural decomposition, arithmetic operators, conditional logic, type conversion, numerical rounding, and scanner-based input processing to model and validate mathematical transformation systems within a command-line environment.
